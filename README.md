<h1>Object'O Pedia</h1>

It is an eCommerce website where you can buy accessories of electronic devices and also some clothing items.

## [Vercel Deployed Link](https://zippy-question-7549.vercel.app/)

## Pages & Features : -
- Home Page

  - Navbar & footer
  - DropDown content
  - Search functionality
  - Card Carousel

- Product Page

  - Filter by category
  - Sort by price
  - Add to Cart
  - Pagination functionality
  - Single ProductPage

- Cart Page

  - Total Summary
  - Discount Coupn
  - Checkout Page
 
- Login & SignUp Page

  - Login Page with Firebase Authentication & Validation
  - Login with google/Email/Phone number
  - Data Store In deployed API and firebase
  - Signup user name displayed on Navbar menu
  - After logged in cart item will displayed on Navbar
  
  
- Admin Page
  
   - Dashboard
   - All the Product summary
   - Logged in User summary
   - Order summary
